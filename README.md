# 2D Platformer Foundation (Unity)
➡️ [На русском](./README.ru.md) || [Download](link)

## Key Features

- Designer-friendly level creation pipeline (zero code)
- Custom Editor tools for level creation
- Addressable-based level loading on launch
- Fully functional main menu and settings
- Cinemachine-based camera
- 2D physics setup (rigidbodies, colliders, layer masks)
- Easy to add new features and mechanics (solid foundation)

## Level Creation Tools

### Grid-based Level Builder
![repo](https://github.com/user-attachments/assets/067e36f9-0392-4658-a498-55ca11e3150a)
Spawns selected prefabs (or fills the whole selected grid with only one)
Allows for faster level creation and iteration

### Platform Path Drawer
![repo1](https://github.com/user-attachments/assets/16ceb96f-b594-4567-b063-e3c35f428692)
Automatically sets up platform's path (point A to point B), with configurable fields (speed, pause)

## Content Pipeline Overview

- Each level is a separate Addressable scene
- Levels are described via ScriptableObjects (name, thumbnail, scene reference)
- Level menu is filled at runtime 
- New levels can be added without modifying code

## Project goals
- Experience full production cycle, from first commit to shippable product
- Provide solid foundation for scalability and further content additions
- Create a designer-friendly content pipeline
